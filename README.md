# R Install

1) __Run command in R:__ devtools::install_github("dombean/lancaster_rmd_posterdown")
2) __Quit RStudio__.
3) __Open RStudio__.
3) __Click:__ File --> New File --> R Markdown. The templates should appear under the package name __cleanRMD__.
